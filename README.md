# AtCoder Practice (C++ for Beginners)

## Overview
AtCoder「C++ for Beginners」の学習項目に沿って、練習問題として提示される問題をC++で解いたコードをまとめています。
コンテストを通しで解くのではなく、学習テーマに対応する問題を選んで取り組んでいます。基本は "cpp.basic" ファイルで保存しているものがメインです。
